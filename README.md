# Descripción del proyecto

Este proyecto integra la API de Open Library para obtener información de libros mediante el uso de fetch en JavaScript. Los datos recuperados son procesados e incorporados a un chatbot interactivo, diseñado para asistir a los usuarios dentro de la plataforma web de Palacio de Tinta.

La página está orientada a un servicio de suscripción de cajas literarias y libros, por lo que el chatbot cumple un rol clave al facilitar la búsqueda, recomendación y acceso a información relevante sobre títulos disponibles, mejorando así la experiencia del usuario.
